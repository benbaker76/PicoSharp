# PicoSharp

**PicoSharp** is an open-source reimplementation of the [PICO-8](https://www.lexaloffle.com/pico-8.php) fantasy console, written in C#/.NET.  
It aims to run PICO-8 cartridges with a focus on readability and portability for desktop systems.

> PicoSharp is not affiliated with Lexaloffle. Please purchase PICO-8 to support the original author.
